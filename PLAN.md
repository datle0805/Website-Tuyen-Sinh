# Project Plan: Admissions Website

## Goal
Design and develop a robust, modern, and user-friendly Admissions Website for prospective students, applicants, and admins.

## Technical Stack
- **Frontend**: Next.js (App Router), TailwindCSS, React Hook Form, Yup.
- **Backend**: Node.js, Express.js, MongoDB.
- **Infrastructure**: Docker.

## Architecture
- **Frontend**: Atomic Design (Atoms, Molecules, Organisms) + Modular features.
- **Backend**: Module-based (Service-Controller-Route).

## Phases
1.  **Setup & Infrastructure**: Initialize projects, Docker, API specs.
2.  **Core Components & Auth**: UI Library, Authentication (JWT).
3.  **Application System**: Database Schema, Multi-step Forms.
4.  **Admin & Review**: Dashboard, Application Management.
